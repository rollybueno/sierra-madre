A modern journal and news block theme with clean typography, CSS based animations, and customizable layouts. Perfect for bloggers, journalists, and content creators.

== Description ==

Chronicle Journal is a modern, block-based WordPress theme designed specifically for journal and news websites. Built with the Full Site Editing (FSE) experience in mind, it combines elegant typography, modern patterns with professional animations to create engaging content experiences.

**Key Features:**

* **Full Site Editing (FSE) Ready** - Complete block theme with Site Editor support
* **Journal-Focused Design** - Optimized for news, blogs, and content-heavy websites
* **Professional Animations** - Smooth hover effects and interactive elements
* **Custom Block Patterns** - 9 pre-built patterns for common journal layouts
* **Responsive Layout** - Mobile-first design with flexible block layouts
* **Developer Friendly** - Well-documented code with comprehensive functions.php documentation

**Design Features:**

* Custom color palette with semantic color names (accent, highlight, alternate)
* Typography system with Playfair Display and Source Sans 3 fonts
* Flexible spacing system with responsive clamp() values
* Professional animations for buttons, links, and interactive elements
* Legacy support for classic WordPress features (galleries, chat format, more tags)
* Enhanced navigation with child menu styling

**Block Patterns Included:**

* Hero Posts Section
* Latest Posts Grid
* Brand Highlights
* Newsletter Signup
* Featured Section
* Promotions Layout
* Team Member Cards
* Brand Partnership
* Latest Posts with Categories

== Installation ==

**From WordPress.org (Recommended):**

1. In your admin panel, go to Appearance → Themes
2. Click "Add New" and search for "Chronicle Journal"
3. Click "Install" and then "Activate"

**Manual Installation:**

1. Download the theme ZIP file
2. In your admin panel, go to Appearance → Themes
3. Click "Add New" → "Upload Theme"
4. Choose the ZIP file and click "Install Now"
5. Click "Activate" to use your new theme

**Requirements:**

* WordPress 6.0 or higher
* PHP 7.4 or higher
* Block theme support enabled

== Frequently Asked Questions ==

= How do I customize the theme? =

Chronicle Journal uses the WordPress Site Editor for customization:

1. Go to Appearance → Editor (Site Editor)
2. Edit templates, template parts, and navigation
3. Customize colors, typography, and layout settings
4. Preview changes in real-time

= Can I use this theme with the Classic Editor? =

Chronicle Journal is designed for the block editor and Site Editor. While it will work with the Classic Editor, you'll get the best experience with the block editor.

= How do I change colors and fonts? =

1. Go to Appearance → Editor
2. Click the Settings icon (gear) in the top right
3. Navigate to "Styles" → "Typography" or "Colors"
4. Customize your theme's design system

= Is this theme mobile-friendly? =

Yes! Chronicle Journal is built with a mobile-first approach and is fully responsive across all devices.

= Can I add custom CSS? =

Yes, you can add custom CSS through:
1. Site Editor → Styles → Additional CSS
2. Or by editing the theme files directly

= Does this theme support custom post types? =

Yes, Chronicle Journal works with all custom post types and supports custom block patterns for specialized content.

= Is this theme translation-ready? =

Yes, Chronicle Journal is fully translation-ready and includes proper text domain support.

= What block patterns are included? =

Chronicle Journal includes 9 custom block patterns:
- Hero Cover Section
- Latest Posts Grid
- Brand Highlights
- Newsletter Signup
- Featured Section
- Promotions Layout
- Team Member Cards
- Brand Partnership
- Latest Posts with Categories

= How do I use the block patterns? =

1. Go to Appearance → Editor
2. Click the "+" button to add a new block
3. Search for "Chronicle Journal" in the patterns section
4. Select and insert your desired pattern
